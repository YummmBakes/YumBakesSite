================================================================================
IF LOGIN FAILS - QUICK FIXES
================================================================================

ISSUE: "Network Error" or "Failed to Fetch"
  FIX: Is backend running? Start backend with F5 in Visual Studio

ISSUE: "CORS Error"
  FIX: Add http://localhost:5173 to appsettings.json Cors.AllowedOrigins
       Then restart backend

ISSUE: Wrong Port
  FIX: Update .env with correct backend port
       Then restart frontend (npm run dev)

ISSUE: User doesn't exist
  FIX: Go to https://localhost:7000/swagger
       Register new user first
       Then try login

For detailed troubleshooting, see: TROUBLESHOOTING_FLOW.md
